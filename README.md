feedparser
==========

Go package for parsing RSS and Atom feeds


INSTALLATION
============

Simply run

	go get github.com/iand/feedparser

Documentation is at [http://go.pkgdoc.org/github.com/iand/feedparser](http://go.pkgdoc.org/github.com/iand/feedparser)
